# 오일 쇼크가 한국 경제 및 금융시장에 미치는 효과 분석
60211477 정민종


## 1. 문제 정의
지정학적 위기로 인한 유가 급등(Oil-shock)이 원유 수입 의존도가 높은 한국의 거시 경제 지표 및 금융시장에 미치는 효과를 분석한다. 
또한, 오일 쇼크와 원인 유형에 따라 한국 경제가 어떻게 반응하는지를 분석한다.

- 데이터 수집 
WTI 원유 선물 가격, 브렌트유 선물 가격, 원/달러 환율, KOSPI 지수, 한국은행 기준금리, 소비자물가지수(CPI)

- 질문
Q1 유가가 n% 이상 급등한 시점을 기준으로, KOSPI지수가 저점을 찍기까지 걸리는 평균 시자(TimeLag)는 얼마인가?
Q2 유가 상승 국면에서 섹터별 수익률 차이는 얼마인가?
Q3 유가 급등이 수입 물가에서 소비자 물가(CPI)로 전이되는 속도(시차)는 얼마인가?
Q4 오일 쇼크의 유형(공급/수요/지정학)에 다른 한국 증시 반응이 통계적으로 유의미하게 다른가?

- 분석 대상 유가 급등 사건
1. 2011년        리비아 내전 - 공급 충격
2. 2014 - 2016년 세일 혁명 유가 급락 - 공급 과잉
3. 2020 - 2022년 코로나 폭락 후 경제 회복 - 수요 충격 -> 공급 부족
4. 2022년        러시아-우크라이나 전쟁 - 지정학 충격


## 2. 기술스택
1. 데이터 수집 : Python
2. 데이터 저장 : HDFS
3. 데이터 전처리 : Apache Spark(PySpark)
4. 데이터 분석 : Apache Hive(HiveQL) + SparkSQL
5. 시각화 : Python
